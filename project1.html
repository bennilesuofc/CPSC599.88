Tubular Touch Synth Controller

Project Description

This device creates sound by interpreting a user’s input via an Arduino and the SuperCollider software. It is played using the five tubes on its front: pressing each tube will play a note, and pushing or pulling the tube will change the pitch of that note. These interactions are detected by photoresistors and slide potentiometers respectively.

My intent with this project was to create a device that could be altered to play many different sounds on a few inputs, like retuning the open strings on a guitar. This device is more like having an array of trombones, where changing the length of each trombone’s slide (aka each tube) changes the pitch.

Different mappings for this device are very possible. There are five digital and analog inputs each, so the user could have a lot of control over the sound produced. For example, the tubes could be used to control different modulation envelopes for a sound in addition to its pitch. SuperCollider is a pretty complex software with a lot of potential for this sort of remapping, but I didn’t explore too deeply into it, mostly because I adopted the software pretty late into the project. I was originally going to have the device be a MIDI controller, but apparently the Arduino UNO has some issues with USB communication vs Serial communication. Also, MIDI didn’t really fit my idea for the user’s interactions with the device.

This project was also fraught with mechanical issues. I wanted to emulate the shortening and lengthening used by analog instruments to change the pitch of notes, and the simplest way to do this seemed to be by using slide potentiometers. Making my own seemed difficult; there would be issues with mounting, stability, and variations in electrical resistance. I decided it would be easier to buy some instead. As it turned out, this wasn’t really the case. The slide pots I bought online were not compatible with my breadboard- their pins were just slightly out of line with the breadboard’s pin holes. As well, all the pins were in-line, so even if I did bend them to fit into the breadboard, they would short out anyway. Rotating them horizontally could’ve worked, but I would only have been able to fit two slide pots at either end of the breadboard, instead of all five in a row like I wanted. I ended up bending the middle wiper pins on the pots so that they stuck out from underneath, and then cut up a paperclip to use as a connection between those and the breadboard. I then stuck pieces of cardboard in between the paperclip bits and the bodies of the slide pots to prevent shorts. It works, somehow.

Another issue I faced was the photoresistors on the ends of the tubes. I initially wanted to make capacitive touch sensors for the ends of the tubes, but this would’ve required more pins on the Arduino than were available. Like with the DIY slide pots, there were also potential issues with different electrical resistances. I decided to instead use photoresistors, as these only require one pin each on the Arduino. However, photoresistors are an analog input, not digital, and all the analog pins were being taken up by the slide pots. Luckily, I found a post online that explained how to set up a voltage divider circuit and the necessary code so that I could use the photoresistors as digital inputs. Some other issues I encountered were: attaching photoresistors to the ends of the tubes, attaching the tubes to the slide pots, and wiring the circuitry so that there was enough room for the tubes to move back and forth.

From these experiences, and from using the device a few times, I came up with some possible changes for a future design:

  • • A tab or handle to make pushing and pulling on the tubes easier
  • • Extra weight in the enclosure so that it won’t slide around when pushing on the tubes
  • • Solder connections instead of using a breadboard- less likely to fall apart, can use irregular components, smaller size overall
  • • Remove photoresistors from inside tubes- don’t have to worry about the movement of the tubes dislodging the photoresistors’ wiring
  • • Longer slide pots- more stable tubes, greater precision in adjustments
