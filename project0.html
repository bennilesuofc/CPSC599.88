RGB Colour Mixer

Project Description

This device allows you to create a colour by adjusting the red, green, and blue levels of an LED with a joystick. Pressing the joystick in creates a random colour.

I had initially wanted to use the joystick to move around in a colour space, with the LED displaying the colour at the current coordinates in the colour space (pressing in the stick would still make a random colour by moving to a random coordinate). Part of this design’s appeal to me was the simplicity- just a single joystick and RGB LED. However, this would have required mapping the joystick’s coordinates to an xyY coordinate system, and then further mapping the result to the RGB values that the LED uses. As well, the LED’s data sheet did not seem to display its colour gamut, so some of the values for the calculations would have to have been guessed at.

I decided to directly adjust each of the LED’s RGB values instead. While I could have used buttons to easily increase or decrease the values, I thought that the analog potentiometers in the joystick would provide finer control to the user. The joystick only has two axes though, so instead of assigning a value to each axis, I made one axis switch between red, blue, and green, and the other axis adjust its value. I also added another RGB LED to the device to indicate to the user which value they are currently adjusting. It would also be possible to switch between adjusting red, blue, and green using a button (or two), but that would’ve required wiring in more components, and one of the axes of the joystick would have gone unused. I also wanted to try to stay close to my original design’s simpler control scheme.
